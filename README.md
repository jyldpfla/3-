# SI 회사 사내 경영 프로그램
- 문제점 : 산발적으로 운영되던 수작업 기반 시스템으로 인한 운영의 비효율성
- 목적 : 인사, 일정, 업무 관리 등 회사 내부 운영에 필요한 모든 과정을 하나의 플랫폼에서 처리해 경영 관리의 효율성 및 업무 생산성 향상을 도모
---
## 프로그램 시연
---

## 주요 기능 및 로직
- 로그인 및 권한 기반 접근 제어
- 사내 일정 관리 및 캘린더
- 사원 개인 업무/이력/담당 내역 전자 프로세스화
- 프로젝트, 팀  통합 관리
- 공지, 사원 문의 등 게시판
- 실시간 알림
